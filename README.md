Portfolio Website (MERN Stack)

A full-stack developer portfolio built using the MERN stack (MongoDB, Express.js, React.js, Node.js). This portfolio showcases my projects, skills, and experience, with a modern UI and a scalable backend.

The application is fully responsive and designed to give visitors a smooth browsing experience across devices. It includes dynamic project data, a contact form, and clean navigation, all powered by a RESTful backend.

Features

Modern and responsive UI built with React

Dynamic project and content management

Backend API using Node.js and Express.js

MongoDB database integration

Contact form with server-side handling

Clean, minimal, and user-focused design

Tech Stack

Frontend

React.js

HTML5

CSS3 / Tailwind CSS (if used)

Backend

Node.js

Express.js

Database

MongoDB

Purpose

This portfolio was created to showcase my full-stack development skills, real-world projects, and understanding of building scalable web applications using the MERN stack.
